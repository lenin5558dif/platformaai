# PlatformaAI - Оценка проекта

## Общая информация

**PlatformaAI** - это полнофункциональная SaaS-платформа для доступа к множеству LLM (Large Language Models) с корпоративными функциями безопасности, биллингом и мультиканальным доступом (Web + Telegram).

## Технологический стек

### Frontend
- Next.js 15 (App Router, Server Components)
- React 19 с TypeScript
- Tailwind CSS 4

### Backend
- Node.js + TypeScript
- Next.js API Routes
- Prisma ORM v5.22

### База данных
- PostgreSQL

### Интеграции
- OpenRouter API (Multi-LLM)
- OpenAI Whisper (Speech-to-Text)
- Stripe (Платежи)
- UniSender (Email/Magic Links)
- Telegram Bot API

## Функционал проекта

### Core Features
1. **Multi-LLM Chat** - поддержка любых моделей через OpenRouter
2. **Streaming responses** - real-time ответы от AI
3. **File Upload** - PDF, DOCX, CSV парсинг
4. **Chat History** - сохранение, поиск, шаринг чатов

### Authentication
5. **Magic Link** - вход по email через UniSender
6. **Telegram Login** - авторизация через Telegram
7. **SSO/OIDC** - корпоративная авторизация

### Billing
8. **Credit System** - кредитная система с балансом
9. **Stripe Integration** - пополнение баланса
10. **Usage Tracking** - учёт токенов и расходов
11. **Spending Limits** - дневные/месячные лимиты

### Enterprise Features
12. **DLP (Data Loss Prevention)** - защита от утечки данных
13. **Model Policy** - контроль доступных моделей
14. **SCIM 2.0** - синхронизация пользователей с IdP
15. **Audit Logging** - полный аудит действий
16. **Organization Management** - управление командой

### Telegram Bot
17. **Text Chat** - полноценный чат через Telegram
18. **Voice Messages** - транскрибация через Whisper
19. **Model Selection** - выбор модели в боте

### Additional
20. **Web Search** - поиск в интернете для контекста
21. **Prompt Templates** - сохранённые промпты
22. **Cost Centers** - центры затрат для биллинга

## Оценка сложности

### Техническая сложность: 8/10

**Высокая сложность обусловлена:**
- Multi-provider LLM интеграция с fallback

- Enterprise-grade DLP и SCIM 2.0
- Сложная биллинговая логика с лимитами
- Мультиканальность (Web + Telegram)
- Полноценный аудит

**Требуется глубокое понимание:**
- LLM рынка и ценообразования

- Биллинговых моделей SaaS
- Интеграций с IdP (SCIM/SSO)
