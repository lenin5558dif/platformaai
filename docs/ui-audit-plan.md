# UI Audit Plan

Дата: 2026-04-16

Цель: привести интерфейс к более цельному product-качеству, проверить размеры кликабельных элементов, hover/focus/cursor, плавность анимаций и согласованность CTA.

Текущий проход: полный `UI`-аудит по `auth/settings`, `layout/admin`, глобальным стилям и ключевым элементам `chat`.

## План

- [x] Проверить глобальные стили и базовые интерактивные состояния: `cursor`, `hover`, `focus`, `disabled`, `prefers-reduced-motion`
- [x] Проверить экран авторизации и onboarding-настройки: размеры кнопок, тексты CTA, читаемость состояний, плотность формы
- [x] Проверить shell, меню пользователя и навигацию: размеры триггеров, hover/focus, мобильное меню
- [x] Проверить админку: таблицы, кнопки действий, destructive actions, компактность и читаемость
- [x] Проверить чат: основные CTA, контраст, hover/focus, размер кликабельных контролов
- [x] Прогнать финальную проверку: typecheck, точечные тесты, UI smoke-check

## Найденные проблемы

- [x] Нет централизованного правила для `cursor: pointer` на интерактивных элементах
- [x] Нет общего снижения/отключения анимаций для `prefers-reduced-motion`
- [x] Есть слишком мелкие secondary CTA в shell/admin, hover/focus стили местами слабые
- [x] Нужно выровнять primary/secondary/destructive кнопки по минимальной высоте и визуальному ритму
